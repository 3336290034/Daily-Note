## 如何遍历一个整数

- 直接上代码

```c
int num = 123;
int g_wei = num % 10; // 个位
int s_wei = (num / 10) % 10; // 十位
int b_wei = (num / 100) % 10; // 百位

while循环重写：
int wei_sum = 0;
while (num)
{
    wei_sum = wei_sum * 10 + num % 10;
    num /= 10; 
}
if (num == wei_sum)
{
    // 是回文数
    // 用这个思路也可以反转整数
}
```
