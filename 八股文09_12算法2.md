## 哈希思想 + 滑动窗口

```c
int lengthOfLongestSubstring3(char *s)
{
    int last[128] = {0};        // 记录每个字符上次出现的位置
    int max = 0;                // 记录最大长度
    int left = 0;               // 记录左指针
    for (int i = 0; s[i] != '\0'; i++)
    {
        if (last[s[i]] > left)
        {
            left = last[s[i]];
        }
        int len = i - left + 1;
        if (len > max)
            max = len;
        last[s[i]] = i + 1; // 记录下一个位置
    }
    return max;
}
```
